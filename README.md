# Nex Utilities

This plugin provides miscellaneous edits/additions related to Nex:
- Mutes blood reavers in the minion room before the bank
- Swaps left-click on the altar to 'Teleport' whilst not in the fight
- Deprioritizes altar use with no zaros item equipped
- Deprioritizes entrance with empty inventory slots
- Opens game chatbox automatically upon kill finish, to include time & mvp in screenshot
